# velocity

Apache Velocity template playground.

## Usage

Edit the [template], and check the [template variables] then run:

    ./gradlew run

## Tests

    ./gradlew test

[template variables]: https://github.com/garriguv/velocity/blob/master/src/main/java/Main.java#L11
